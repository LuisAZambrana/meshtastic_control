# Meshtastic Control by surc.ar
Minimalist IRC-style Meshtastic Client Terminal for:

- Raspberry Pi
- Linux
- Windows

## Features

- Multi-tab chat
- Direct Messages (DM)
- IRC-style interface
- Meshtastic serial support
- Node browser
- Channel support
- LoRa presets
- Cross-platform serial detection

## Installation

Requires Python 3.13

```bash
pip install meshtastic-control
